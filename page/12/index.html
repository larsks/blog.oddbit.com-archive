  7. Why does the Neutron documentation recommend three interfaces?

    The documentation for configuring Neutron recommends that a network controller has three physical interfaces: Before you start, set up a machine to be a dedicated network node. Dedicated network nodes should have the following NICs: the management NIC (called MGMT_INTERFACE), the data NIC (called DATA_INTERFACE), and the external NIC (called EXTERNAL_INTERFACE). People occasionally ask, “why three interfaces?
